# вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/bagade-paras/Algomentor-Ai.git
```

Move into the project

```bash
cd Algomentor-Ai
```

---

## Install Frontend

```bash
cd client
npm install
```

Run frontend

```bash
npm start
```

---

## Install Backend

```bash
cd ../server
npm install
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run backend

```bash
npm start
```
