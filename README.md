# 🚀 AlgoMentor AI

An AI-powered Data Structures & Algorithms learning platform built with the MERN stack and Google's Gemini AI. Users can ask DSA-related questions and receive detailed, AI-generated explanations to improve problem-solving and interview preparation.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/bagade-paras/Algomentor-Ai.git
```

Move into the project

```bash
cd Algomentor-Ai
```

---

## Install Frontend

```bash
cd client
npm install
```

Run frontend

```bash
npm start
```

---

## Install Backend

```bash
cd ../server
npm install
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run backend

```bash
npm start
```
